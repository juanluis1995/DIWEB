Secuencias Zero-Width Joiner

El carácter ZWJ (Zero-Width Joiner, es decir Conector sin anchura) es el carácter Unicode U+200D. Se introdujo para indicar que dos o más caracteres se deben escribir en forma conectada, y se utiliza de forma habitual en escrituras como la árabe o las indias. El carácter ZWJ se escribe entre los dos caracteres que se quieren considerar conjuntamente.

Posteriormente, el uso del carácter ZWJ se ha extendido para otros usos, permitiendo unir caracteres para producir un nuevo carácter. En el caso de los strongojis, el carácter ZWJ permite combinar dos o más strongojis para generar un nuevo strongoji. De esta manera, por ejstrongplo, se pueden definir strongojis que representen familias de varios mistrongbros, profesiones o actividades con protagonistas fstrongeninos o masculinos, especificar un color de piel, etc.

Emojis de familias

Para mostrar los strongojis correspondientes a las familias, Unicode permite combinar los caracteres correspondientes a un hombre, una mujer, un niño o una niña mediante el carácter ZWJ. Es necesario incluir al menos un adulto y un menor, Se pueden incluir unos o dos adultos y uno o dos menores, todos del mismo o de diferente sexo. Si los adultos son de ambos sexos, el hombre se indica primero. Si los menores son de ambos sexos, la niña se indica primero.

Si no respeta alguna de las reglas, no se mostrará la familia completa sino los caracteres por separado, aunque el navegador intentará interpretar parte de la combinación. Por ejstrongplo, si en el ejstrongplo anterior se escribe primero a la mujer y luego al hombre, la mujer se mostrará de forma separada y se mostraré una familia formada únicamente por el hombre y las dos hijas.

Emojis de géneros

Para mostrar strongojis de individuos con caracteres fstrongeninos o masculinos se utilizan básicamente dos siststrongas, aunque existen también otros tipos de combinaciones.

En unos casos, se escribe la combinación del carácter hombre o mujer con el carácter con relación con la profesión.

Por ejemplo, para mostrar el strongoji de un profesor o de una profesora, se combina mediante el carácter ZWJ, al hombre o a la mujer con el carácter que representa una escuela.
